PROJECT 4

Recipe Finder

The Recipe Finder App helps users find new and exciting recipes based on what they search for. Using the Edamam API, it lets users enter keywords like ingredients, dishes, or types of cuisine to find recipes. The app shows results with images, titles, ingredient lists, and links to detailed instructions. It displays a set number of recipes per page (4 by default) and allows users to navigate through multiple pages easily. Built with HTML and CSS for design and JavaScript for functionality, it uses asynchronous data fetching for a smooth experience. It also has error messages for when no recipes are found or if there is a problem fetching data. This project highlights my skills in JavaScript, working with APIs, and creating responsive designs.
